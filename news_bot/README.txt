Описание проекта
Проект представляет собой Telegram-бот для просмотра новостей с различных источников. Бот позволяет:

Просматривать новости из источников: "Коммерсант", "РИА Новости", "РБК", "CNN".
Настроить источник новостей по умолчанию.
Настроить количество выдаваемых новостных заголовков за раз.
Получать последние новости или все доступные новости.
Функционал бота включает:

Главное меню с кнопками "НОВОСТИ" и "НАСТРОЙКИ".
Подменю новостей с возможностью выбора всех новостей, последних новостей или конкретного источника.
Меню настроек для выбора источника по умолчанию и количества новостей.
Настройки пользователя сохраняются в JSON-файле, что обеспечивает персонализацию для каждого пользователя.

Инструкция по запуску
1. Установка зависимостей
Для работы бота необходимо установить библиотеку aiogram и другие зависимости. Выполните команду:

pip install aiogram

2. Создание конфигурационного файла
Создайте файл config.py в корневой директории проекта с содержимым:

BOT_TOKEN = "YOUR_BOT_TOKEN"
Замените YOUR_BOT_TOKEN на токен вашего Telegram-бота, который можно получить у BotFather .

3. Создание файла парсеров новостей
Создайте файл pars.py в корневой директории проекта. В этом файле должны быть реализованы функции для парсинга
новостей с различных источников:

def parse_kommersant():
    # Логика парсинга новостей с "Коммерсанта"
    return [{"title": "Заголовок новости", "link": "https://example.com"}]

def parse_ria():
    # Логика парсинга новостей с "РИА Новости"
    return [{"title": "Заголовок новости", "link": "https://example.com"}]

def parse_rbc():
    # Логика парсинга новостей с "РБК"
    return [{"title": "Заголовок новости", "link": "https://example.com"}]

def parse_cnn():
    # Логика парсинга новостей с "CNN"
    return [{"title": "Заголовок новости", "link": "https://example.com"}]


4. Запуск бота
Запустите бота, выполнив команду:

python bot.py
Бот начнет работать, и вы сможете взаимодействовать с ним через Telegram.


Схема базы данных (JSON-файл)
Настройки пользователей хранятся в файле user_settings.json. Файл имеет следующую структуру:

{
    "USER_ID_1": {
        "default_source": "Коммерсант",
        "news_limit": 5
    },
    "USER_ID_2": {
        "default_source": "РБК",
        "news_limit": 10
    }
}

Описание полей:
USER_ID : Уникальный идентификатор пользователя в Telegram (строка).
default_source : Источник новостей по умолчанию (строка). Возможные значения: "Коммерсант", "РИА Новости", "РБК", "CNN".
news_limit : Количество новостей, которое бот выдает за раз (целое число).

Дополнительные замечания
Парсинг новостей :
Для реализации парсеров можно использовать библиотеку requests для HTTP-запросов и BeautifulSoup (из пакета bs4) для
 парсинга HTML.
Убедитесь, что парсеры соответствуют правилам использования API или сайта.
Обработка ошибок :
Если источник новостей недоступен, бот должен выводить сообщение об ошибке.
Добавьте проверку на случай, если файл user_settings.json поврежден или отсутствует.
Масштабирование :
Если количество пользователей увеличится, рекомендуется перейти на использование базы данных (например, SQLite или
PostgreSQL) вместо JSON-файла.

Заключение
Этот проект предоставляет удобный способ получения новостей через Telegram-бота с возможностью настройки под
индивидуальные предпочтения. Благодаря простоте реализации и использованию JSON-файла для хранения настроек, проект
легко адаптируется под новые источники новостей или дополнительные функции.
